# 🌍 VoyaGo – Travel Management System

A full-stack Travel Management System built using the MERN stack that enables users to explore, plan, and manage travel bookings with secure authentication and email-based workflows.

---

## 🚀 Live Demo

* 🌐 Frontend: https://voya-go-trvel.vercel.app

---

## 🧰 Tech Stack

* **Frontend:** React.js (Vite), HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas
* **Authentication:** JWT (JSON Web Tokens)
* **Email Service:** Nodemailer (Gmail SMTP)
* **Deployment:** Vercel (Frontend), Render (Backend)

---

## ✨ Features

* 🔐 User Authentication (Register/Login)
* 🔑 Forgot & Reset Password (Email-based)
* ✈️ Flight Booking System
* 📦 RESTful API Integration
* 🔒 Secure Password Hashing (bcrypt)
* 🌐 Fully Deployed Cloud Application
