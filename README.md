# 🌍 Travel Management System

A full-stack web application that helps users plan, organize, and manage their travel experiences efficiently. Built using the MERN stack, this system provides a seamless interface for trip booking, itinerary management, and destination exploration.

## 🚀 Features

- 🔐 User Authentication (Login/Register with JWT)
- 🧳 Trip Planning & Management
- 📍 Explore Destinations
- 📅 Itinerary Scheduling
- 💳 Booking Management
- 🛠️ Admin Panel for managing users, trips, and bookings
- 📱 Fully Responsive UI

## 🏗️ Tech Stack

### Frontend:
- React.js
- HTML5, CSS3, JavaScript

### Backend:
- Node.js
- Express.js

### Database:
- MongoDB (NoSQL Database)
- Mongoose (ODM for MongoDB)
